# Data-Analytics-Internship-Task2
# Sales Data Analysis and Business Intelligence Dashboard

## Project Overview

This project was completed as part of the ApexPlanet Internship Task 2.

The objective of the project is to perform Exploratory Data Analysis (EDA), answer business questions using SQL, and create a dashboard to visualize key business insights.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* SQL (SQLite)
* Google Colab
* Excel / Power BI

## Project Files

### EDA_Notebook.ipynb

Contains:

* Data loading
* Data cleaning
* Descriptive statistics
* Histogram
* Scatter plot
* Correlation heatmap
* Business insights

### EDA_Report.pdf

Detailed report containing:

* Dataset overview
* Statistical analysis
* Visualizations
* SQL findings
* Business conclusions

### SQL_Queries.md

Contains SQL business questions and their results.

### Dashboard.png

Static dashboard displaying:

* Total Orders
* Average Sales
* Revenue by Product
* Profit by Product
* Top Customers by Sales
* Revenue vs Profit

## Key Findings

* Webcam generated the highest revenue.
* Webcam generated the highest profit.
* Rohit was the top customer based on sales.
* Average sales per order was 27,357.95.
* Revenue and profit showed a positive relationship.

## Conclusion

The project successfully identified top-performing products, valuable customers, and profitability trends through data analysis and visualization techniques.
